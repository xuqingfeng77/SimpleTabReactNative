# SimpleTabReactNative
主要效果：react-native-tab-navigator，已经banner的使用<br>
如果其他环境都配置好可以执行下面命令安装<br>
npm install --save react react-native   安装react native<br>
npm install react-native-tab-navigator --save  安装tab插件<br>
